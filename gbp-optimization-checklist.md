# Google Business Profile Checklist (Week 3)

## Profile setup
- Business name exactly matches site branding: NOVA BODA
- Website URL set to canonical domain
- Primary category selected
- 2-3 secondary categories added
- Service areas include Valencia + nearby municipalities

## Conversion elements
- Business description includes key terms naturally:
  - organizador de bodas en Valencia
  - proveedores de boda en Valencia
- Phone/email visible and working
- Messaging enabled (if team can respond quickly)
- Appointment/contact CTA configured

## Trust and relevance
- Upload 15-25 high-quality photos:
  - planning moments
  - vendor coordination
  - event details
- Add weekly post:
  - one tip + one guide link + one CTA
- Add FAQs and seed first Q&A items

## Reviews process
- Create short review request script
- Ask every satisfied lead/provider
- Respond to every review in <=72h
- Include city/context in responses naturally

## Tracking
- Record weekly:
  - profile views
  - website clicks
  - calls
  - direction requests
- Compare against Search Console and Analytics trends
